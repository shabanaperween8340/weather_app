A "Weather App" is a user friendly application designed to provide real-time and accurate weather information . Users can access current conditions , hourly forecasts , and extended outlooks for various locations . The app utilizes reliable weather APIs to deliver data on temperature , humidity , wind speed . With an intuitive interface , interactive maps , and personalized notifications , the weather app enhances users' ability to plan their activities based on up-to-date meteorological insights . It aims to offer a seamless and informative experience , catering to users' diverse weather-related needs.

Software Requirement:-
Operating System : Windows 11
Web-Browser: Internet Explorer , chrome , Microsoft Edge .

Technology Used : Html , CSS,Javascript

Source Code Editor : VS Code.
